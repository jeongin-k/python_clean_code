깨끗한 코드는 한 가지를 제대로 한다.

깨끗한 코드는 잘 쓴 문장처럼 익힌다.

특정 목적을 달성하는 방법은(여러가지가 아니라) 하나만 제공한다.

깨끗한 코드는 언제나 누군가 주의 깊게 짯다는 느낌을 준다.

중복 줄이기, 표현력 높이기, 조반부터 간단한 추상화 고려하기

코드가 그 문제를 풀기 위한 언어처럼 보인다면 아름다운 코드라 불러도 되겠다.
